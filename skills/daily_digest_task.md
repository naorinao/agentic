Daily digest task:

- Review the fetched payload and decide if it is important enough to send to Slack.
- If you notify Slack, write a plain-text daily report with the title `Team Daily Report`.
- Use a single section headed `Today's Work`.
- Under that heading, include 3 to 8 bullets when the data supports it.
- Each bullet should include the action, the object worked on, and the specific outcome.
- Do not over-compress details into vague summary language.
- When the source data comes from GitHub, include the GitHub URL in each relevant bullet.
